rbTabs

Description

<rb-tabs> is a directive for navigation tabs.

Accepts a comma + whitespace separated list of tab names and either a string path to the directory with template files or a callback when tabs are switched Templates should match with tab names, in lower case and hyphenated.

If a callback is provided, it will return the tab which has been clicked and activated back to the controller

Usage

<!-- It will look for the following templates
    path/to/tabs/tab1.html
    path/to/tabs/tab2.html
    path/to/tabs/tab-3.html
-->

<rb-tabs tabs="tab1, Tab2, TAB 3" tab-src="path/to/tabs"></rb-tabs>

<!-- Changing tabs will call the switchTabs(tab) callback -->

<rb-tabs tabs="tab1, tab2, tab3"
    tab-callback="switchTabs(tab)">
</rb-tabs>


<!-- tab-src and tab-callback can be used together -->

<rb-tabs tabs="tab1, tab2, tab3"
    [tab-callback="switchTabs(tab)"]
    [tab-src="path/to/tab/templates"]
    [tab-options="arrayOnScope"]>
<rb-tabs>

Parameters

ParamTypeDetails
tabs
(optional)
string

List of tab names

tab-options
(optional)
array

Array with tabs config:

 [
     {
         name: 'nameOfTab',
         slug: 'tabHeading',
         uiSref: 'linkOfTab',
     }
 ]
tab-src
(optional)
string

Path to directory which contains tab templates

active-tab
(optional)
string

Default active tab, otherwise the first tab in the list will be active

tab-classes
(optional)
string

List of classes to be applied to each individual tab

container-classes
(optional)
string

List of classes to be applied to tab content container

state-param
(optional)
string

Which $stateParam should directive listen to for the active tab

heading
(optional)
boolean

Applies heading classes, defaults to true

page
(optional)
boolean

Applies nav-page class, defaults to false

tab-callback
(optional)
Function

Function on controller scope which is to be called when tab is switched
