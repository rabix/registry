rbDropdown

Description

<rb-dropdown> is a dropdown button directive.

Default button look is 'default'

Usage

<!-- Simple dropdown, name can be two way binded from parent ctrl -->
<rb-dropdown intention="primary" name="view.dropdown">
  <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
  </ul>
</rb-dropdown>

<!-- Split button dropdown -->
<rb-dropdown intention="primary" type="split" name="'Split dropdown'">
  <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
  </ul>
</rb-dropdown>

<!-- Disabled dropdown -->
<rb-dropdown intention="primary" type="split" name="'Disabled'" ng-disabled="true">
  <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
  </ul>
</rb-dropdown>

Parameters

ParamTypeDetails
nameexpression

label of dropdown button, can be two way binded, can pass in a string: name="'Some name'"

intention
(optional)
boolean

If present, sets button intention oneOf=['danger', 'primary', 'default', 'warning', 'info', 'success'].

ng-disabled
(optional)
expression

En/Disable based on the expression

type
(optional)
string

Dropdown type, oneOf: ['split', 'default']. Default value is 'default'

iconClass
(optional)
string

FontAwesome icon class which is prepended to the name. No fa- prefix necessary.

class
(optional)
string

Space separated additional classes to add to element

position
(optional)
string

To which side the dropdown opens, oneOf: ['left', 'right']. Default value is 'left'
